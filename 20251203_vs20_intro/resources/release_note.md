We are excited to announce that Vertex AI Vector Search 2.0 (VS2.0) is now in Public Preview.

This release introduces a powerful, simple-to-use retrieval engine designed to streamline AI development and serve as the knowledge core for your AI applications (e.g. RAG, Search). 

VS2.0 lets you go from 0 to prototype in less than 5 minutes, and seamlessly scale to production.

What is Vector Search 2.0?

Vector Search 2.0 creates a powerful, semantic retrieval engine for any AI application with minimal lines of code. It provides a clear path from raw data to rich, contextual answers, handling the complexities of embedding and indexing without any infra work.

Designed from the ground up, VS2.0 offers a radically simplified developer experience that allows you to move from 0 to production-scale retrieval in minutes.

Key use cases: RAG, semantic and/or keyword search, recommendation engines, agentic retrieval, image/video search, ad serving, and many more!
